Introduction

Today marks the first day of my journey to becoming a Full Stack Web Developer. I finally committed to learning web development seriously and started with the fundamentals of HTML. It felt exciting to build something from scratch using just code!

What I Learned

I began with understanding the structure of a webpage and how browsers read HTML. Here are some of the key concepts I learned today:

  • Basic HTML structure including <html>, <head> and <body>
  • Text formatting tags and links
  • Semantic HTML tags like <header>, <main>, <section>, and <footer>

And here’s a sample structure I practiced today:


<header>
   <h1>Welcome to My Website</h1>
</header>

Challenges

No learning is complete without challenges. I found it tricky to remember which tags are semantic and which are not. But I found a trick to remember:

“If a tag describes the purpose of a section of content, it’s semantic.”

But with practice, I know this will become second nature.

Goals

My plan for Day 2 is to begin learning CSS. My goals are:

  1. Complete the basics of CSS
  2. Build a styled version of my portfolio page
  3. Learn how Flexbox works
